## 🔐 凯撒密码详解

凯撒密码是最简单的替换密码，以下用数学语言描述：

设明文字母表为 $\Sigma = \{A, B, C, ..., Z\}$，密文字母表为 $\Sigma' = \{D, E, F, ..., Z, A, B, C\}$。

加密函数：$E(x) = (x + 3) \mod 26$
解密函数：$D(x) = (x - 3) \mod 26$

### 正则表达式表示

可以用正则表达式描述凯撒密码的模式：
- 原文字母：`[A-Z]`
- 加密后字母：`[D-ZA-C]` （向后移3位）

## 🧮 数学原理

### 模运算基础

模运算在密码学中至关重要：

$(a + b) \mod m = ((a \mod m) + (b \mod m)) \mod m$

$(a \times b) \mod m = ((a \mod m) \times (b \mod m)) \mod m$

### 欧几里得算法

求最大公约数：$gcd(a,b) = gcd(b, a \mod b)$

```python
def gcd(a, b):
    while b != 0:
        a, b = b, a % b
    return a
```

## 🔑 现代加密算法

### AES (高级加密标准)

AES是一种对称加密算法，使用SPN (Substitution-Permutation Network)结构。

**数学基础：**
- 有限域 $GF(2^8)$ 上的运算
- 字节代替：通过S盒实现非线性变换
- 行移位：提供扩散效果
- 列混合：线性变换矩阵

### RSA算法

RSA基于数论难题：大整数分解。

**密钥生成：**
1. 选择两个大素数 $p, q$
2. 计算 $n = p \times q$
3. 计算 $\phi(n) = (p-1)(q-1)$
4. 选择公钥 $e$，满足 $gcd(e, \phi(n)) = 1$
5. 计算私钥 $d = e^{-1} \mod \phi(n)$
